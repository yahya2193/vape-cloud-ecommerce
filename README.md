# Vape Cloud E-Commerce Frontend

## Project Description
A fully responsive, modern e-commerce web application built using React and React-Bootstrap. The frontend features a dynamic home page landing carousel, a structured shop page loading product variations, and an interactive shopping cart component managing real-time calculations.

## Technologies Used
* Frontend Framework: ReactJS
* Styling Engine: React-Bootstrap / CSS3
* Routing: React-Router-Dom
* Deployment Host: Vercel Production Pipeline

## Setup Instructions
1. Clone the repository or download the source files.
2. Navigate to the root directory where `package.json` is located.
3. Install dependencies by running: `npm install`
4. Start the local development environment using: `npm start`
5. Open your browser and navigate to `http://localhost:3000`
